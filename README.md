# Pacman Maze

A classic web-based Pacman experience built with HTML5, CSS3, and JavaScript. Navigate the maze, collect dots, and enjoy the retro arcade atmosphere.

## 🚀 Features

* **Interactive Gameplay:** Smooth navigation through custom-designed mazes.
* **Visual Assets:** High-quality SVG maze designs and Pacman sprites.
* **Audio Integration:** Classic sound effects for eating dots and game transitions.
* **Clean Code:** Modular JavaScript structure for easy maintenance.

## 🛠️ Technologies Used

* **HTML5:** Page structure and game container.
* **CSS3:** Layout styling and animations.
* **JavaScript (ES6+):** Game logic, collision detection, and sound management.
